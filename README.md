
# Aplicación de ventas de consolas de video juegos 

## Descripción del Proyecto
Proyecto realizado para un ecommmerce que se dedica a la venta de consolas de videojuegos. 

## Tecnologias utilizadas
El proyecto fue desarrollado empleando las bibliotecas React, React-DOM y React-Router-DOM. La información se almacenó en una base de datos creada con Firebase en su versión 10.3.1. En cuanto a la apariencia visual del proyecto, se utilizó el preprocesador de CSS SASS junto con el framework de estilos CSS Tailwind. Además, se implementaron notificaciones para interactuar con el usuario mediante la librería Toast.

## Usos e instalaciones
npm install react@18.2.0 react-dom@18.2.0
npm install react-router-dom@6.15.0
npm install react-scripts@5.0.1
npm install sonner@0.7.1
npm install firebase@10.3.1
